El programa se encarga de crear una conexión a la base de datos con el nombre y la contraseña indicada en el código. 
Creamos un listado inicial de los 5 primeros ciclistas junto a sus respectivos equipos según su rendimiento y después una comparativa de puntos entre los diferentes equipos. 
También se debe mostrar las etapas que están por encima de la media de distancia, la mas larga o corta y alguna si tiene a mas de 10 ciclistas con mas de 0 puntos.
